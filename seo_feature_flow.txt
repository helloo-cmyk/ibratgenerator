# Ibrat Generator - Core System Architecture, Feature Flows & SEO Implementation Guide

This document serves as the absolute source of truth for the **Ibrat Generator** project (`ibratgenerator.com`). It details the system's core technical stack, global layout configuration, analytics pipelines, interactive editor workflows, page-by-page feature distributions, SEO meta tags, and structured JSON-LD schemas.

---

## 1. Core Technology Stack & Typography
- **Core Framework**: React 19 & Next.js 15 (App Router, optimized with TypeScript).
- **Styling Paradigm**: Hybrid styling using utility-first classes (Tailwind CSS) integrated with raw Vanilla CSS in `globals.css` with a standardized `.hp-` prefix convention for marketing sections.
- **Typography & Font Optimization**: Injected via Next.js Google Fonts optimization in `layout.tsx` to prevent cumulative layout shift (CLS):
  - **Geist Sans** (`Geist`): Loaded as `--font-geist-sans` (subsets: latin, display: swap).
  - **Geist Mono** (`Geist_Mono`): Loaded as `--font-geist-mono` (subsets: latin, display: swap).
  - **DM Serif Display** (`DM_Serif_Display`): Loaded as `--font-display` (weight: 400, subsets: latin, display: swap).
  - **Google Fonts CDN Link**: An additional external stylesheet is injected within the generator component to enable fonts like *Anton*, *Bebas Neue*, *Montserrat*, *Playfair Display*, and *Poppins* dynamically inside the `<canvas>`.
- **Text Standards**: All copy utilizes standard hyphens and commas instead of em dashes (`—`) to strictly follow clean, SEO-friendly body copy guidelines.

## 4. The Interactive Brat Generator Widget (`src/components/BratGenerator.tsx`)
The centerpiece of the application is a highly optimized, client-side canvas designer supporting drag-and-drop operations, real-time font adjustments, custom uploads, sticker overlays, and dynamic clipboard rendering.

### A. Core Interactive Features & Editor Flow
The generator divides controls into three focused categories, adapted for desktop and mobile viewport configurations:
1. **Text Control Tab**:
   - **Text Input**: Multi-line `<textarea>` backing instant Canvas updates on keypress events.
   - **Font Size Slider**: Controls the type scale dynamically between `24px` and `220px` (default: `180px`).
   - **Line Spacing Slider**: Controls height ratios between `0.8` and `1.6` (default: `1.0`, step: `0.05`).
   - **Letter Spacing Slider**: Controls typographic tracking between `-4` and `10` (default: `0`, step: `0.5`).
   - **Align Selector**: Sets paragraph alignments to `left`, `center`, or `right`.
   - **Font Family Selector**: Allows users to select between *Inter* (default), *Poppins*, *Montserrat*, *Bebas Neue*, *Anton*, and *Playfair Display*.
   - **Border Outline System**: Toggleable text stroke outline with a dedicated custom Color Picker.
   - **Drop Shadow System**: Toggleable soft dropshadow underneath text elements with a custom Color Picker.
2. **Style & Color Tab**:
   - **Preset Swatches**: Instantly sets layout parameters:
     - *Brat*: Background `#c1ff00`, Text `#0a0a0a`
     - *Deluxe*: Background `#0a0a0a`, Text `#ffffff`
     - *Crash*: Background `#ff90e8`, Text `#0a0a0a`
     - *White*: Background `#ffffff`, Text `#0a0a0a`
   - **Background & Text Color Pickers**: Full custom hex picker control.
   - **Live Contrast Checker**: Evaluates color contrast between text and background on-the-fly (`#brat-contrastLabel`).
   - **Aspect Ratio Presets**: Adjusts dimensions to `1:1` (Square), `4:5` (Instagram), `9:16` (Story/Reel), or `16:9` (Wide).
   - **Resolution Presets**: Selectable exports: `1024px`, `1500px` (default), `2048px`, or `3000px`.
   - **Safe Margins Guidelines**: Toggleable dashed border guides to help align compositions.
   - **Selfie Mode (Image Backgrounds)**: Allows local image file uploads (`#brat-bgUpload`). The uploaded photo covers the background automatically while text and stickers remain active layers on top. Includes a dedicated removal trigger.
3. **Stickers & Emoji Tab**:
   - **Category Switcher**: Allows sorting stickers by category groups.
   - **Sticker Interaction Grid**: Direct insertion of emojis/stickers onto the canvas viewport.
   - **Sticker Mouse & Touch Gestures**:
     - *Single Tap/Click*: Places the sticker.
     - *Drag*: Moves stickers around the canvas viewport.
     - *Corners Handle Drag*: Dynamically scales the sticker size.
     - *Top Handle Drag*: Rotates the sticker.
     - *Double Click/Double Tap*: Deletes the target sticker.

### B. Action Controls & Output Exporters
- **Undo / Redo Managers**: High-fidelity history stack.
- **Copy Image**: Directly serializes the active `<canvas>` state to the system clipboard (`navigator.clipboard.write`) for instant copy-pasting.
- **Save PNG**: Renders a dedicated high-fidelity graphic at the designated resolution preset (up to 3000px), bypassing compression, and triggering an immediate browser download.

### C. Desktop vs. Mobile Layout Orchestration
- **Desktop Vibe (width >= 1000px)**: Two-column layout where controls sit in a sticky side panel (`#brat-controls`, width: `380px`) and the interactive canvas remains pinned in a high-contrast viewport stage. Uses top-aligned tabs.
- **Mobile Vibe (width < 1000px)**: App-like layout featuring:
  - Sticky bottom navigation bar (`.brat-bottom-nav`) with intuitive emoji highlights.
  - Floating Action Button (FAB) or high-visibility bottom pill to trigger the Save PNG action instantly.
  - Interactive bottom drawer configuration (`#brat-controls` with `.sheet-open`) incorporating a touch-friendly grab handle indicator.
  - Dynamic responsive scaling of the `<canvas>` viewport (`height: 95vw`) to ensure high usability on smaller mobile touchscreens.
