# Ibrat Generator - Complete Feature Flow and SEO Implementation Inventory

## 1) Core Product Feature Flow (End-to-End)

1. User opens any tool page and lands in hero + live generator.
2. User enters text in textarea (`#brat-text`) with instant preview rendering.
3. User customizes typography:
   - Font family (`Inter`, `Poppins`, `Montserrat`, `Bebas Neue`, `Anton`, `Playfair Display`)
   - Font size, line height, letter spacing
   - Text align (`left`, `center`, `right`)
   - Optional outline + outline color
   - Optional soft shadow + shadow color
4. User customizes colors:
   - Preset pills (`brat`, `deluxe`, `crash`, `white`)
   - Manual text/background color pickers
   - Contrast readout with AA hint
5. User customizes canvas:
   - Aspect ratio: `1:1`, `4:5`, `9:16`, `16:9`
   - Export resolution: `1024`, `1500`, `2048`, `3000`
   - Safe margins toggle
6. User can upload/remove background photo in selfie mode.
7. User can add stickers/emojis by category and interact with them:
   - Place, drag, resize, rotate, delete (double click or handle)
8. User can transform text directly on canvas:
   - Drag, rotate handle, resize handles
9. User uses undo/redo (history stack with redo stack support).
10. User exports:
    - Save PNG (watermark-free)
    - Copy image to clipboard (with fallback behavior)
    - Optional batch ZIP export via JSZip + FileSaver

## 2) Generator Interaction and UX Systems

- Desktop tabs: `Text`, `Style`, `Stickers`
- Mobile app-like bottom nav + sticky controls sheet
- Sticky preview panel on desktop
- Keyboard and pointer interaction handling
- Touch-friendly hit areas and long-press delete on touch
- Toast notifications for copy/export feedback
- Dynamic/lazy load of generator (`ssr: false`) through `BratGeneratorLazy`
